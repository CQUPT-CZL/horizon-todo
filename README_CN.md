## 📸 视觉预览 (UI Gallery)

<p align="center">
  <img src="public/web.png" width="90%" style="border-radius: 15px; box-shadow: 0 10px 30px rgba(0,0,0,0.1);" alt="Horizon Todo UI" />
</p>

> **设计理念：** 摆脱传统列表的束缚，用灵动的卡片布局（Sector Grid）重新定义任务管理。

---

## ✨ 核心特性

- 🧩 **Sector Grid 布局**：打破常规，让待办事项像星辰一样散落在画布上。
- 🌈 **优先级色彩方案**：`Urgent` (紧急), `Focus` (聚焦), `Normal` (常规) 三档视觉区分。
- 🍃 **极简交互**：清爽的输入框设计，专注于“记录”本身。
- ☁️ **持续集成**：基于 Vercel 自动部署，代码提交即更新。

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/CQUPT-CZL/horizon-todo.git

# 进入目录
cd horizon-todo

# 安装依赖
npm install

# 本地运行
npm run dev
```
